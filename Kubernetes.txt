How to create a cluster using shell script -> faster than manual creating



->create an instance-> connect ssh terminal
->sudo su -
->hostnamectl set-hostname master_node.example.com/worker_node1.example.com
->create a editor 
->vim kubernetes_master_node.sh
->copy paste the control plane script/worker script(if making worker node) in the vim editor -> save it
->now give permission to execute the file
->chmod +x kubernetes_master_node.sh
->run the file 
->./kubernetes_master_node.sh

->scripts have been downloaded 

How to deploy a pod ?


vim my-first-pod.yaml
apiVersion:v1
Kind: Pod
metadata:
name: devops
labels:
**app: nginx**

spec:
containers:
      **- name: web-app**

	**image: nginx:1.14**

	**ports:**

	   **- containerPort: 80**







kubectl apply -f my-firstpod.yaml
kubectl get pod
kubectl descrive pod devopscp


kubectl describe pod prod-pod | grep -i controlled
