Q2-> How to change policies / activate & deactivate a user / lock & unlock a user ?
	
Ans-> 
Policies for a user can be changed using two method :-

	1> chage -d 5 -E 10 -I 3 -m 30 -M 99 -W 15 user_name 

	{ d->date of last changed password, I-> period of inactivity, E-> expiration date m->minimum date required, M-> maximum date required , w->warn}													
	
	2> vim /etc/login.defs  change the minimum , maximum and warn days and the policies will be permanently changed -> i-insert ; w-> save ; q-> quit 
	3> cat /etc/shadow or cat /etc/passwd to see policies changed or not 


Activating and Deactivating a user :-

	1>usermod -s /sbin/nologin user_name -> try accesing this user using root or other user it should show error means its deactivarted
	2>usermod -s /bin/bash user_name ->try accessing now it should be accessible its activating

Lock and Unlock a user :-

	1>usermod -L user_name
	2> usermod -U user_name


Q3-> How to create a group / add users to a group/multiple groups / change group/ownership/permission of user ?

Ans->
create groups :-
	1>groupadd group_name ->will create the group
	2>vim /etc/groups   -> to see your groups you have created  vim /etc/group | grep -i user -> to find if a user belongs to a group or not

Add users to a group :-
	1>usermod -G group_name user_name

Add a user to multiple groups :-
	1>usermod -aG group_name user_name

changing group ownership of a file  :-
	1>chgrp group_name file_name

changing permissions of a file :-
	1>chmod 754 file/folder/directory_name {7->root all permission , 5-> groups { permission to execute and read } ,5->others {permission to read only}

changing user ownership of a file/directory :-
	1>chown user_name file_name
